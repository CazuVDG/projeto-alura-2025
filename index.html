Calculadora Interativa

Este projeto foi meu primeiro contato com JavaScript e lógica de programação aplicada a interfaces web. Desenvolvi uma calculadora que realiza operações básicas como soma, subtração, multiplicação e divisão.

Utilizei HTML para criar a estrutura da calculadora, com botões para números e operações. O CSS foi usado para estilizar a interface, dando um visual moderno e agradável.

Com JavaScript, implementei a lógica da calculadora, aprendendo sobre:

  • Manipulação do DOM para capturar cliques nos botões
  • Armazenamento de valores em variáveis
  • Estruturas condicionais para determinar a operação
  • Funções para realizar os cálculos

Este projeto foi fundamental para entender como JavaScript pode tornar páginas web interativas e dinâmicas, respondendo às ações do usuário.

Jogo da Memória

Meu primeiro jogo desenvolvido com JavaScript puro. Um jogo da memória com cartas temáticas sobre tecnologia, onde o objetivo é encontrar todos os pares no menor número de tentativas.

Neste projeto, apliquei vários conceitos importantes de programação:

  • Manipulação de arrays para armazenar e embaralhar as cartas
  • Event listeners para capturar cliques nas cartas
  • Lógica para comparar cartas e verificar pares
  • Contagem de movimentos e tempo de jogo
  • Feedback visual para o jogador

O maior desafio foi implementar a lógica do jogo, especialmente a parte de virar as cartas, verificar os pares e determinar quando o jogo termina. Também adicionei um contador de tentativas e um temporizador para tornar o jogo mais desafiador.

Este projeto me mostrou como a programação pode ser divertida e criativa, além de ter me ajudado a consolidar conceitos fundamentais de JavaScript.
